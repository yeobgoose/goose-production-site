# Production 신청 전 체크리스트

## 1. 웹사이트
- [ ] 본인이 소유·관리하는 실제 도메인에 HTTPS로 배포
- [ ] 홈페이지 정상 접속
- [ ] 모바일/PC에서 주요 화면 확인
- [ ] 데모 탭 3개 동작 확인
- [ ] Discord Bot 초대 링크 실제 주소로 교체
- [ ] 문의 이메일 실제 주소로 교체
- [ ] 개인정보처리방침 실제 운영 내용으로 수정
- [ ] 이용약관 실제 운영 내용으로 수정
- [ ] Riot 일반 정책의 "최신 필수 legal boilerplate"를 공식 정책 페이지에서 다시 확인해 사이트에 정확히 반영
- [ ] Riot 로고를 임의로 사용하지 않았는지 확인
- [ ] 서비스가 Riot 공식 제품처럼 보이지 않는지 확인

## 2. 제품 설명
- [ ] MMR/ELO 대체 서비스가 아님을 명시
- [ ] Impact Score / Play DNA는 비공식 자체 지표라고 명시
- [ ] 경기 종료 후 분석 중심임을 명시
- [ ] 익명 플레이어 식별 기능이 없음을 확인
- [ ] 숨겨진 실시간 정보로 경쟁 우위를 제공하지 않음을 확인
- [ ] Production 신청 설명과 실제 배포 기능이 일치

## 3. Riot API
- [ ] 신청 API 목록이 실제 코드와 일치
- [ ] API Key를 프론트엔드 코드에 넣지 않음
- [ ] API Key는 서버 환경변수로만 관리
- [ ] HTTPS 사용
- [ ] 제품 1개당 Production Key 1개 원칙 확인
- [ ] 공개 서비스에서 Development Key를 사용하지 않음

## 4. Production 신청
- [ ] Riot Developer Portal에서 제품 등록
- [ ] Product URL 입력
- [ ] Terms URL 입력
- [ ] Privacy URL 입력
- [ ] 동작하는 웹 프로토타입 링크 입력
- [ ] 필요 시 데모 영상/스크린샷 추가
- [ ] 사용자 흐름 설명 입력
- [ ] 신청 후 Riot이 제공하는 사이트 인증 문자열/URL 확인
- [ ] 지정된 URL에 인증 문자열 업로드
- [ ] Verified 상태 확인

## 5. 승인 후
- [ ] `.env`에 Production API Key 등록
- [ ] `/등록` 실제 테스트
- [ ] `/최근경기` 실제 테스트
- [ ] `/실제복기` 실제 테스트
- [ ] `/자동복기 켜기` 실제 테스트
- [ ] 중복 경기 처리 확인
- [ ] AI fallback 확인
- [ ] 개인정보 삭제 요청 처리 절차 확인
- [ ] 24시간 운영 서버로 배포
