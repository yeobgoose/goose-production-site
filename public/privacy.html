1. 개요

협곡도사 덕구(이하 “서비스”)는 Discord를 통해 League of Legends 경기 종료 후 복기와 성장 추적 기능을 제공하는 독립 서비스입니다. 본 방침은 서비스가 어떤 정보를 처리하고 왜 사용하는지 설명합니다.

2. 처리하는 정보

  • Discord 사용자 ID, 서버 ID 및 자동복기 알림 채널 ID
  • 사용자가 직접 등록한 Riot ID(게임 이름 및 태그라인)
  • Riot API를 통해 연결된 PUUID 및 경기 식별자
  • 경기 결과, 참가자 통계, Match 및 Timeline 기반 분석 데이터
  • 서비스가 생성한 복기, 플레이 DNA, 반복 실수 패턴 및 미션 진행 기록
  • 오류 진단에 필요한 최소한의 서버 로그

3. 이용 목적

  • 사용자 계정과 Riot 계정 연결
  • 경기 종료 후 복기 및 통계 분석 제공
  • 장기 성장 지표와 미션 진행도 저장
  • 사용자가 요청한 Discord 채널로 자동복기 알림 전송
  • 오류 탐지, 보안 및 서비스 안정성 유지

4. AI 기능

AI 코칭 기능이 활성화된 경우 복기에 필요한 일부 게임 통계와 분석 결과가 외부 AI API로 전송될 수 있습니다. 서비스 설계상 Discord 사용자 ID, PUUID, Riot API 키와 같은 인증·직접 식별용 값은 AI 코칭 요청 본문에 포함하지 않는 것을 원칙으로 합니다.

5. 보관 및 삭제

경기 분석과 성장 기록은 기능 제공을 위해 보관될 수 있습니다. 사용자는 운영자에게 계정 연결 해제 또는 저장 데이터 삭제를 요청할 수 있습니다. 구체적인 보관 기간은 정식 운영 정책 확정 후 본 방침에 반영합니다.

6. 제3자 서비스

서비스는 기능 제공을 위해 Discord, Riot Games API 및 선택적으로 AI API 제공자의 서비스를 이용할 수 있습니다. 각 외부 서비스의 처리에는 해당 제공자의 약관 및 개인정보 관련 정책이 적용될 수 있습니다.
